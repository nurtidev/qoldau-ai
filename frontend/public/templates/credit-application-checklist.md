# Чек-лист документов для кредитной заявки

Используйте этот список, чтобы подготовить полный пакет документов перед подачей заявки на финансирование.

## Правоустанавливающие документы
- [ ] Устав / учредительный договор (для юридических лиц)
- [ ] Свидетельство о государственной регистрации (БИН/ИИН)
- [ ] Решение/протокол о назначении руководителя
- [ ] Доверенность (если заявку подаёт представитель)
- [ ] Копия удостоверения личности заявителя/руководителя

## Финансовая документация
- [ ] Бухгалтерская отчётность за последние 2-3 года
- [ ] Налоговая декларация за последний период
- [ ] Справка об оборотах по расчётным счетам за 12 месяцев
- [ ] Справка о наличии/отсутствии действующих кредитов и лизинга
- [ ] Расшифровка дебиторской и кредиторской задолженности

## Документы по проекту
- [ ] Бизнес-план или технико-экономическое обоснование
- [ ] Финансовая модель проекта
- [ ] Коммерческие предложения на оборудование/материалы (если применимо)
- [ ] Договоры с ключевыми поставщиками и покупателями (при наличии)

## Обеспечение (залог)
- [ ] Правоустанавливающие документы на предмет залога
- [ ] Отчёт об оценке залогового имущества
- [ ] Справка об отсутствии обременений

## Дополнительно
- [ ] Справка об отсутствии задолженности по налогам и обязательным платежам
- [ ] Согласие на запрос кредитной истории
- [ ] Реквизиты банковского счёта для перечисления средств

Совет: подготовьте сканы документов в формате PDF, названия файлов — на латинице, без пробелов.
